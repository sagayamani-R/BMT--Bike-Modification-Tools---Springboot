🏍️ Bike Modifications Tools (BMT) Platform
Bike Modifications Tools (BMT) is a secure, intelligent, and scalable next-generation e-commerce marketplace specifically tailored for motorcycle enthusiasts to purchase accessories, spares, and riding gear.

Built using a robust Spring Boot and PostgreSQL backend paired with a responsive Thymeleaf and Bootstrap frontend, the platform seamlessly converges advanced cryptographic security, artificial intelligence, and tiered system governance.

🚀 Key Features
Multi-Tiered Role-Based Access Control (RBAC): Distinct operational environments for Users (storefront browsing & shopping), Admins (catalog, inventory, and order management), and Super Admins via the BMT ADMINS dashboard (payment tracking, API log monitoring, and admin delegation).

Dual-Layered Cryptographic Security: Implements symmetric AES Encryption to safeguard sensitive user transaction data alongside BCrypt Hashing (with salting and adaptive complexity) to protect user credentials against brute-force attacks.

AI-Powered Customer Experience: Features an integrated Spring AI Chatbot for personalized product recommendations and query resolution, complemented by an AI-powered Intelligent Search Bar utilizing semantic analysis for dynamic category and keyword suggestions.

Production-Ready Architecture: Designed using modular Spring Boot microservices and optimized PostgreSQL indexing to sustain high concurrent traffic and seamless horizontal scalability.

🛠️ Tech Stack & Tools
Backend: Spring Boot, Java, Maven, Spring AI

Database: PostgreSQL

Frontend: Thymeleaf, HTML5, CSS3, JavaScript, Bootstrap

Security: AES Encryption, BCrypt Hashing, HTTPS, Token-based Session Management
